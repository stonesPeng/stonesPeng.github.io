Docker CE 分为 stable test 和 nightly 三个更新频道。每六个月发布一个 stable 版本 (18.09, 19.03, 19.09…)，生产环境使用stable版本。

1.1 卸载旧版本Docker,Docker-Engine

b.重启Docker服务，done

若不想设置全局的镜像源，也可在拉取镜像时指定镜像源
